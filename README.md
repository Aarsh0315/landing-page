# 🚀 Karigarr Clothing - Landing Page

This project is a **responsive landing page** built as part of **CodSoft Internship (Task 2)**.

## 📌 Project Overview

Karigarr Clothing is a modern landing page designed to showcase products with a clean and user-friendly interface. The focus of this project is on layout design, responsiveness, and UI structure using HTML, CSS, and Bootstrap.

---

## ✨ Features

* Responsive Navigation Bar
* Bootstrap Carousel (Hero Section)
* Product Cards Section (3-column layout)
* About Section
* Clean Footer Design
* Hover Effects & UI Styling

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
